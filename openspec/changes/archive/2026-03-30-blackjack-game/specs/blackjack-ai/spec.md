## 新增需求

### 需求：AI 庄家必须遵循 Hard 17 规则
AI 庄家在所有玩家完成行动后，必须持续摸牌直至点数 ≥ 17（包含 Soft 17 时可选配置）。

#### 场景：庄家点数 ≤ 16 时必须摸牌
- **当** AI 庄家当前手牌点数 ≤ 16
- **那么** 系统必须自动为庄家执行 Hit，间隔 600ms 以呈现视觉动画

#### 场景：庄家点数 ≥ 17 时必须停牌
- **当** AI 庄家当前手牌点数 ≥ 17
- **那么** 系统必须自动为庄家执行 Stand，进入结算阶段

#### 场景：Soft 17 配置
- **当** `GameConfig.dealerHitSoft17` 为 true 且庄家手牌为 Soft 17（含 A 计 11 的 17 点）
- **那么** 系统必须为庄家执行 Hit；若该配置为 false，则 Stand

### 需求：AI 庄家暗牌在结算前必须对玩家隐藏
庄家发牌时第二张为暗牌，系统必须隐藏暗牌点数直至庄家行动阶段开始。

#### 场景：暗牌隐藏
- **当** 游戏处于玩家行动阶段
- **那么** 庄家第二张牌必须显示为牌背，点数显示为"?"

#### 场景：暗牌翻开
- **当** 所有玩家完成行动，进入庄家行动阶段
- **那么** 系统必须翻开庄家暗牌，显示真实点数并更新庄家总点数
