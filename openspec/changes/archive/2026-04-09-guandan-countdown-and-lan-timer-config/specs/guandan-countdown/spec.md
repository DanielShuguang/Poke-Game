## ADDED Requirements

### 需求：掼蛋出牌倒计时显示
掼蛋游戏页面在轮到本地玩家行动时（出牌/进贡/还贡），必须在按钮区域上方显示倒计时标签。倒计时初始值必须取自页面的 `turnTimeLimit` 参数（默认 35 秒）。剩余时间 ≤ 10 秒时，标签颜色必须变为红色（`dangerRed`）；> 10 秒时为琥珀色（`accentAmber`）。

#### 场景：出牌阶段显示倒计时
- **当** 游戏处于 playing 阶段且轮到本地玩家出牌
- **那么** 按钮区域上方显示倒计时标签，格式为「N 秒」，每秒递减

#### 场景：进贡/还贡阶段显示倒计时
- **当** 游戏处于 tribute 或 returnTribute 阶段且轮到本地玩家操作
- **那么** 按钮区域上方显示倒计时标签

#### 场景：倒计时颜色切换
- **当** 剩余时间从 11 秒变为 10 秒
- **那么** 标签背景色从琥珀色变为红色

#### 场景：玩家操作后倒计时消失
- **当** 玩家出牌、不出或完成进贡/还贡
- **那么** 倒计时立即停止并从 UI 中消失

#### 场景：轮到 AI 时无倒计时
- **当** 轮到 AI 玩家行动
- **那么** 不显示倒计时

### 需求：掼蛋单机模式超时自动托管
单机模式下，本地玩家倒计时归零时，系统必须自动代为执行操作。出牌阶段：首出时调用 `forcePlayCards`（出最小牌），跟牌时调用 `forcePass`（不出）。进贡阶段：自动选择手牌中最大的非鬼牌进贡。还贡阶段：自动选择手牌中最小的牌还贡。

#### 场景：出牌阶段首出超时
- **当** 单机模式下本地玩家首出，倒计时归零
- **那么** 系统调用 `forcePlayCards` 自动出最小合法牌组

#### 场景：出牌阶段跟牌超时
- **当** 单机模式下本地玩家跟牌，倒计时归零
- **那么** 系统调用 `forcePass` 自动不出

#### 场景：进贡阶段超时
- **当** 单机模式下本地玩家需要进贡，倒计时归零
- **那么** 系统自动选择手牌中 rank 最大的非鬼牌进贡

#### 场景：联机模式不自动托管
- **当** 联机模式下倒计时归零
- **那么** 页面不自动执行操作（由 Host 的 NetworkAdapter 超时托管）

### 需求：掼蛋页面接受 turnTimeLimit 参数
`GuandanGamePage` 必须新增 `int turnTimeLimit` 可选构造参数，默认值 35。倒计时和超时托管必须使用此参数值。

#### 场景：使用默认时限
- **当** 以 `GuandanGamePage()` 创建页面（不传 turnTimeLimit）
- **那么** 倒计时初始值为 35 秒

#### 场景：使用自定义时限
- **当** 以 `GuandanGamePage(turnTimeLimit: 15)` 创建页面
- **那么** 倒计时初始值为 15 秒
