## 修改需求

### 需求：游戏选择页展示可用游戏
游戏选择页必须展示所有可用游戏的入口卡片。每张卡片必须显示游戏名称、简短描述和游戏缩略图。玩家点击卡片后必须进入对应游戏的配置或大厅页面。

#### 场景：展示斗地主入口
- **当** 玩家进入游戏选择页
- **那么** 页面显示斗地主游戏卡片，包含名称和描述

#### 场景：展示德州扑克入口（新增）
- **当** 玩家进入游戏选择页
- **那么** 页面同时显示德州扑克游戏卡片，卡片标注"现金局"模式，点击后进入德州扑克大厅页面

#### 场景：点击进入游戏
- **当** 玩家点击任意游戏卡片
- **那么** 系统导航至对应游戏的房间大厅或模式选择页面
