## 新增需求

### 需求：牌型定义与合法性判断
系统必须识别并验证以下掼蛋牌型；非以下牌型的出牌组合禁止提交。

#### 场景：单张合法
- **当** 玩家提交 1 张牌
- **那么** 系统判定为合法单张

#### 场景：对子合法
- **当** 玩家提交 2 张点数相同的牌
- **那么** 系统判定为合法对子

#### 场景：三张合法
- **当** 玩家提交 3 张点数相同的牌
- **那么** 系统判定为合法三张

#### 场景：三带二合法
- **当** 玩家提交 3 张相同点数 + 1 对相同点数的牌
- **那么** 系统判定为合法三带二

#### 场景：顺子合法（无级牌）
- **当** 玩家提交 5 张及以上连续点数的不同花色单牌（A 不可作为顺子延伸）
- **那么** 系统判定为合法顺子

#### 场景：连对合法
- **当** 玩家提交 3 对及以上连续点数的对牌
- **那么** 系统判定为合法连对

#### 场景：钢板合法
- **当** 玩家提交 3 组及以上连续点数的三张
- **那么** 系统判定为合法钢板

#### 场景：普通炸弹合法
- **当** 玩家提交 4 张及以上相同点数的牌（非级牌）
- **那么** 系统判定为合法炸弹，张数多的炸弹更大

#### 场景：同花顺炸弹合法
- **当** 玩家提交 5 张及以上相同花色且连续点数的牌
- **那么** 系统判定为合法同花顺炸弹，优先级高于同张数普通炸弹

#### 场景：天王炸合法
- **当** 玩家提交 2 张大王
- **那么** 系统判定为天王炸，优先级最高，可压制任何炸弹

#### 场景：非法牌型
- **当** 玩家提交的牌不符合上述任何牌型
- **那么** 系统拒绝出牌，提示"牌型不合法"

---

### 需求：级牌百搭嵌入顺子
系统必须允许当前轮级牌（不含大小王）充当顺子中的缺口牌；同一张级牌禁止在同一顺子中充当多个缺口。

#### 场景：级牌填补单缺口
- **当** 玩家提交的牌组包含连续点数但中间缺 1 个点数，且提交了 1 张对应级牌
- **那么** 系统判定该牌组为合法顺子

#### 场景：级牌不可延伸两端
- **当** 玩家提交的牌组使用级牌作为顺子两端的延伸
- **那么** 系统拒绝该牌型

#### 场景：级牌数量不足以填补缺口
- **当** 玩家提交的牌组有 2 个缺口但只有 1 张级牌
- **那么** 系统拒绝该牌型

---

### 需求：炸弹大小比较
系统必须按以下优先级比较炸弹大小：天王炸 > 同花顺炸（按张数）> 级牌炸（按张数）> 普通炸（按张数）；相同类型按张数比较，张数相同时无法压制。

#### 场景：天王炸压制一切
- **当** 场上已有任意炸弹，玩家出天王炸
- **那么** 天王炸合法压制

#### 场景：同花顺炸压制普通炸
- **当** 场上有普通四张炸弹，玩家出五张同花顺
- **那么** 同花顺炸合法压制

#### 场景：相同类型比张数
- **当** 场上有五张普通炸弹，玩家出六张普通炸弹（相同点数）
- **那么** 六张炸弹合法压制

---

### 需求：升降级结算
每局结束时系统必须根据头游、二游归属计算升降级档数，并更新双方队伍的当前级牌。

#### 场景：己方包揽头游二游，升 2 级
- **当** 同队两人分别是本局第一、第二出完手牌的玩家
- **那么** 己方升 2 级（当前级牌 +2，最高到 A）

#### 场景：己方头游、对方二游，升 1 级
- **当** 己方一人第一出完，对方一人第二出完
- **那么** 己方升 1 级

#### 场景：对方头游、己方二游，不升级
- **当** 对方一人第一出完，己方一人第二出完
- **那么** 双方均不升级，下局换对方先出

#### 场景：对方包揽头游二游，降级
- **当** 对方两人分别是第一、第二出完手牌
- **那么** 己方降 2 级（当前级牌 -2，最低到 2）

#### 场景：升到 A 后继续赢，游戏结束
- **当** 己方当前级牌为 A 且再次满足升级条件
- **那么** 系统宣告己方获胜，游戏结束

---

### 需求：贡牌与还贡流程
若某队上局对方包揽头游二游，下局开始前输方必须向赢方进贡最大手牌，赢方必须还贡一张牌；贡牌非本人手牌中最大单张禁止提交。

#### 场景：进贡合法
- **当** 负方玩家选择并提交手中最大的单张牌
- **那么** 系统接受贡牌并转移给对应赢家

#### 场景：进贡不合法
- **当** 负方玩家提交的贡牌不是手中最大单张
- **那么** 系统拒绝并提示"必须进贡最大牌"

#### 场景：还贡完成后开始出牌
- **当** 所有贡牌与还贡均完成
- **那么** 系统进入正常出牌阶段，由赢方（头游玩家）先出
