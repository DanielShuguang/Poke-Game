# 升级游戏引擎规范

## 新增需求

### 需求：系统必须支持 108 张牌
升级游戏必须使用两副标准扑克牌（52×2=104 张）加 4 张大小王，共 108 张牌。

#### 场景：发牌完成
- **当** 发牌阶段结束
- **那么** 系统必须已分发 100 张牌给 4 名玩家（每人 25 张），保留 8 张作为底牌

#### 场景：牌堆完整
- **当** 游戏初始化
- **那么** 牌堆必须包含所有 108 张牌，每种花色 A-K 各 8 张，大小王各 2 张

### 需求：系统必须实现将牌判定
系统必须根据当前级牌和叫牌花色正确判定将牌。

#### 场景：级牌判定
- **当** 当前级别为 2
- **那么** 所有花色的 2 必须被视为级牌（将牌的一部分）

#### 场景：将牌花色判定
- **当** 叫牌花色为黑桃
- **那么** 所有黑桃花色的牌必须被视为将牌

#### 场景：无将模式
- **当** 叫牌为大小王对子（无将）
- **那么** 仅级牌和大小王为将牌，无将牌花色

#### 场景：将牌大小排序
- **当** 比较将牌大小
- **那么** 排序必须为：大王 > 小王 > 将牌花色级牌 > 其他花色级牌 > 将牌花色 A > K > ... > 2

### 需求：系统必须实现叫牌机制
系统必须允许玩家通过亮牌叫牌确定庄家和将牌。

#### 场景：对子叫牌
- **当** 玩家手中有两张相同点数的牌
- **那么** 玩家可以选择亮出该对子进行叫牌

#### 场景：拖拉机叫牌
- **当** 玩家手中有连续对子（如 77-88）
- **那么** 玩家可以选择亮出拖拉机进行叫牌，优先级高于对子

#### 场景：大小王叫无将
- **当** 玩家手中有大王对子或小王对子
- **那么** 玩家可以选择叫无将

#### 场景：叫牌优先级
- **当** 多名玩家同时叫牌
- **那么** 系统必须按拖拉机 > 对子 > 单张优先级判断，同级别比较点数

### 需求：系统必须验证出牌合法性
系统必须验证玩家出牌是否符合规则。

#### 场景：首出任意牌型
- **当** 玩家为首出者
- **那么** 玩家可以出任意合法牌型（单张、对子、拖拉机）

#### 场景：跟花色规则
- **当** 首出为红桃单张，玩家手中有红桃
- **那么** 玩家必须出红桃，禁止出其他花色

#### 场景：无花色可垫牌
- **当** 首出为红桃，玩家手中无红桃
- **那么** 玩家可以出其他花色（垫牌）或将牌（杀牌）

#### 场景：杀牌大于垫牌
- **当** 玩家 A 出黑桃 A，玩家 B 垫红桃 K，玩家 C 杀大王
- **那么** 玩家 C 必须赢得此轮

#### 场景：拖拉机牌数匹配
- **当** 首出为两对拖拉机
- **那么** 跟牌玩家必须出相同数量的牌（4 张）

### 需求：系统必须正确计算得分
系统必须统计每轮出牌中的分值牌并累计得分。

#### 场景：分值牌统计
- **当** 一轮出牌包含 5、10、K
- **那么** 系统必须计算 5 分 + 10 分 + 10 分 = 25 分

#### 场景：得分归属
- **当** 玩家 A 出牌最大赢得此轮
- **那么** 此轮所有分值牌必须计入玩家 A 所在队伍的得分

#### 场景：总分 200 分
- **当** 所有牌出完
- **那么** 两队得分总和必须为 200 分（每副牌 100 分 × 2）

### 需求：系统必须判定升级结果
系统必须根据最终得分判定升级结果。

#### 场景：大光
- **当** 防守队得 0 分
- **那么** 庄家队必须连升 3 级

#### 场景：小光
- **当** 防守队得分 ≤ 40 分
- **那么** 庄家队必须连升 2 级

#### 场景：成功
- **当** 防守队得分 41-80 分
- **那么** 庄家队必须升 1 级

#### 场景：下台
- **当** 防守队得分 81-160 分
- **那么** 庄家队下台，防守队成为新庄家

#### 场景：大反
- **当** 防守队得分 ≥ 161 分
- **那么** 防守队上台并连升 2 级

### 需求：系统必须识别拖拉机
系统必须正确识别连续对子组成的拖拉机。

#### 场景：基本拖拉机
- **当** 玩家出 77-88（两对连续）
- **那么** 系统必须识别为拖拉机

#### 场景：长拖拉机
- **当** 玩家出 33-44-55-66（四对连续）
- **那么** 系统必须识别为拖拉机

#### 场景：非连续对子
- **当** 玩家出 77-99（不连续）
- **那么** 系统必须识别为两个独立对子，不是拖拉机

#### 场景：将牌拖拉机
- **当** 当前级牌为 2，玩家出 22-33（将牌花色）
- **那么** 系统必须识别为将牌拖拉机
