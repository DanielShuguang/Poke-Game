# 游戏选择规范

## 新增需求

### 需求：房主可以选择游戏类型
系统必须允许房主在创建房间时从游戏列表中选择对战类型。

#### 场景：选择斗地主
- **当** 房主选择游戏类型"斗地主"
- **那么** 系统必须固定房间人数为 3 人，禁用人数配置

#### 场景：选择德州扑克
- **当** 房主选择游戏类型"德州扑克"
- **那么** 系统必须允许房主设置人数范围（2-9 人）

#### 场景：游戏类型未实现
- **当** 房主选择的游戏类型尚未开发
- **那么** 系统必须显示"敬请期待"标签并禁用选择

### 需求：系统必须验证人数配置
系统必须根据游戏规则验证并限制房间人数配置。

#### 场景：斗地主固定人数
- **当** 游戏类型为"斗地主"
- **那么** 系统必须强制房间人数为 3 人，忽略房主的配置

#### 场景：德州扑克人数限制
- **当** 游戏类型为"德州扑克"且房主设置人数为 10 人
- **那么** 系统必须拒绝配置并提示"德州扑克最多支持 9 人"

### 需求：房主可以配置游戏规则
系统必须允许房主根据游戏类型配置特定规则。

#### 场景：斗地主规则配置
- **当** 房主选择"斗地主"
- **那么** 系统必须提供选项：是否允许明牌、底分倍数（1/2/3）

#### 场景：德州扑克规则配置
- **当** 房主选择"德州扑克"
- **那么** 系统必须提供选项：初始筹码数量、小盲注/大盲注比例

### 需求：玩家可以查看游戏规则
系统必须允许玩家在等待大厅查看当前游戏的规则配置。

#### 场景：查看规则
- **当** 玩家在等待大厅点击"查看规则"按钮
- **那么** 系统必须显示当前游戏的规则详情（如底分倍数、盲注设置）

### 需求：游戏配置修改需全员确认
系统必须确保房主修改游戏配置后，所有已准备玩家需重新确认。

#### 场景：修改配置
- **当** 房主修改游戏规则（如底分倍数从 1 改为 2）
- **那么** 系统必须重置所有玩家的"准备"状态，并通知"房主已修改规则，请重新准备"

### 需求：游戏选择页展示可用游戏
游戏选择页必须展示所有可用游戏的入口卡片。每张卡片必须显示游戏名称、简短描述和游戏缩略图。玩家点击卡片后必须进入对应游戏的配置或大厅页面。

#### 场景：展示斗地主入口
- **当** 玩家进入游戏选择页
- **那么** 页面显示斗地主游戏卡片，包含名称和描述

#### 场景：展示德州扑克入口
- **当** 玩家进入游戏选择页
- **那么** 页面同时显示德州扑克游戏卡片，卡片标注"现金局"模式，点击后进入德州扑克大厅页面

#### 场景：点击进入游戏
- **当** 玩家点击任意游戏卡片
- **那么** 系统导航至对应游戏的房间大厅或模式选择页面

### 需求：游戏选择页新增炸金花入口
系统必须在游戏选择页面中展示炸金花游戏卡片，允许用户进入炸金花游戏。

#### 场景：显示炸金花入口
- **当** 用户在主页游戏选择区域浏览
- **那么** 系统显示炸金花游戏卡片，包含游戏名称、图标和简短描述

#### 场景：进入炸金花游戏
- **当** 用户点击炸金花游戏卡片
- **那么** 系统导航至炸金花游戏页面

## 新增需求

### 需求：游戏选择页展示 21 点入口
首页游戏列表必须包含 21 点游戏卡片，显示游戏名称、简介，以及与斗地主/德州扑克/炸金花相同的"联机对战"快速按钮。

#### 场景：展示 21 点卡片
- **当** 玩家进入首页
- **那么** 游戏列表必须显示 21 点卡片，包含"开始游戏"和"联机对战"两个按钮

#### 场景：点击联机对战
- **当** 玩家点击 21 点卡片的"联机对战"按钮
- **那么** 系统必须跳转至 `/room/scan?gameType=blackjack`，过滤并显示 21 点房间列表
#### 场景：展示牛牛卡片
- **当** 玩家进入首页
- **那么** 游戏列表必须显示牛牛卡片，状态为 `available`，包含"开始游戏"和"联机对战"两个按钮

#### 场景：点击牛牛联机对战
- **当** 玩家点击牛牛卡片的"联机对战"按钮
- **那么** 系统必须跳转至 `/room/scan?gameType=niuniu`，过滤并显示牛牛房间列表

### 需求：游戏选择页展示升级入口
首页游戏列表必须包含升级游戏卡片，显示游戏名称、简介，以及与其他游戏相同的"联机对战"快速按钮。

#### 场景：展示升级卡片
- **当** 玩家进入首页
- **那么** 游戏列表必须显示升级卡片，包含"开始游戏"和"联机对战"两个按钮

#### 场景：点击升级联机对战
- **当** 玩家点击升级卡片的"联机对战"按钮
- **那么** 系统必须跳转至 `/room/scan?gameType=shengji`，过滤并显示升级房间列表

#### 场景：升级卡片显示固定人数
- **当** 玩家查看升级卡片
- **那么** 卡片必须显示"4人组队"标识
